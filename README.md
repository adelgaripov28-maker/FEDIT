# FEDIT - Reddit Clone

A simple Reddit-like forum built with pure HTML, CSS, and JavaScript.

## Features
- User registration and login
- Create posts in different communities
- Upvote/downvote posts
- Responsive design

## Demo
Visit: [your-username.github.io/fedit-gh-pages](https://your-username.github.io/fedit-gh-pages)

## Local Development
1. Clone the repository
2. Open `index.html` in your browser
3. Or use a local server: `python3 -m http.server 8000`
